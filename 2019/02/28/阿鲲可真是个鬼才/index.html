阿鲲可真是个鬼才

发布 : 2019-02-28

几个月之前看完《风味人间》，令我影像深刻的有三样东西：镜头、文案、主题曲。结合了现在高科技技术，以及完美的镜头，在观众眼前，不仅仅吸引观众的眼球，也抓住了观众的感受。美食与故事之间很难以结合，但是有了文案就不一样了，《风味人间》的文案就像是把美食的灵魂抓出来，这个灵魂就是要讲的故事。还是就是主题曲《那些美好》，镜头和文案很可能让人遗忘，而音乐却不会。

循环细听，这首歌是人生，小号响起正如生命的诞生，紧接的欢快正如彩色的童年，接着节奏逐渐放缓上扬正如人生的上升和沉淀，之后又加入降调比喻人生起伏，之后钢琴的适时加入峰回路转直到平静且安详的收官；这首歌更是人间，恢弘大气的主旋律是人类历史发展的车轮，细节的婉转则更多的是富有烟火气的柴米油盐。（这段文案摘自QQ音乐，看到写得好就拿过来用了）

上礼拜看了《流浪地球》，影片结束的时候竟然看到阿鲲参与部分OST。感叹这个人真的鬼才，能通过《那些美好》讲述人生，也能通过《开启新征程》挑战未来。
